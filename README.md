# Cisco-conf
Mise en place d'une infrastructure réseau pour le BTS SIO


-HSRP, attendus : Tolérance de pannes et redondance en cas de panne d’un routeur.  \
-STP, attendus : redondance et suppression de boucles.  \
-VLAN par ports et routage inter-vlan, attendus : isolation d’un réseau et communication avec les autres réseaux.  \
-Configuration de plages DHCP et agent relais DHCP, attendus : fournir à nos machines clientes des adresses IP, masques et passerelles par défaut.  \
-RIPv2, attendus : routage dynamique pour aller au-delà du réseau d’interconnexion.  \
-EtherChannel, attendus : redondance en cas de panne d’un lien.  \
